运算符 · 模块脚本

一、运算符 · 逻辑

核心类别对应函数作用内容
布尔逻辑and, or, not, xor对布尔值或数值执行逻辑 与 / 或 / 非 / 异或 运算，支持标量、向量、矩阵；对应符号 &&、||、!、^^。
位运算bitAnd, bitOr, bitXor对整型数值按位执行 AND / OR / XOR 运算，对应符号 &、|、^；也可用于集合的交集 / 并集操作。

二、运算符 · 关系

核心类别对应函数作用内容
基本比较eq, ne, gt, ge, lt, le对两个对象进行 等于 / 不等 / 大于 / 大于等于 / 小于 / 小于等于 的比较，返回布尔值；对应符号 ==、!=、>、>=、<、<=。
浮点与对象比较eqFloat, eqObj, eqPercenteqFloat 在指定精度下比较浮点数；eqObj 对任意类型对象做深度相等判断；eqPercent 在允许误差百分比内判断两值是否相等。
范围与成员between, inbetween 判断值是否在 [lower, upper] 闭区间内；in 判断值是否属于给定集合，支持向量批量查找。

三、模块管理

核心类别对应函数作用内容
模块安装与加载installModule补从远程或本地源安装模块到模块目录，便于后续在会话中使用。
模块枚举listRemoteModules补列出远程可安装模块，便于查看可用模块清单。
已加载模块查询getLoadedModules补查询当前会话或系统已加载的模块列表，用于确认模块是否已生效。

四、脚本 / 运行时

核心类别对应函数作用内容
脚本执行run补, runSQL补, runScript补执行字符串脚本、SQL 语句或脚本文件，是运行时动态执行入口。
调试与测试test补, terminate补test 用于测试脚本或函数；terminate 用于终止运行中的任务或会话。
输出与拼接print补, sqlTuple补print 输出调试信息；sqlTuple 构造 SQL 片段/元组，便于脚本化拼装。
工作目录getExecDir补, getWorkDir补分别获取可执行文件目录与当前工作目录，常用于文件路径定位与脚本环境检查。
