# DolphinDB Easily-Finding · 函数速查手册

> 一套依照官方文档重构的 DolphinDB 全函数总览,按使用场景重新归类,帮助使用者快速定位目标函数。
>
> 在线访问:**[ccaaff.github.io/DolphinDB_Function_Quick_Reference_Guide](https://ccaaff.github.io/DolphinDB_Function_Quick_Reference_Guide)**

---

## 缘起:干中学的实践

经过笔者「干中学」的实践,得到一条朴素的理论:

> 告诉你「香蕉沾辣椒」之后,你才知道有这道地狱美食的食谱。

函数的使用同理 —— **如果没有对应高级函数的意识,就根本不知道还有这种用法**。

为了更好、更有逻辑地总览 DolphinDB 的全部函数,本项目依照官方已发布的文档信息,把以下两份内容相互对照:

- 文档侧边栏的「全部函数」列表
- DolphinDB 官方「函数分类」

在此基础上,**补齐了分类页面中缺失、但官方实际已发布的函数**,并对页面进行重构,方便使用者快速定位目标函数。

目前部分函数类别重构较为成功,内容仍在持续打磨中。
