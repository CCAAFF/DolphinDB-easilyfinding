一、元编程

核心类别对应函数作用内容
表达式构造与解析expr, parseExpr, evalexpr 把对象或字符串转换为表达式对象；parseExpr 将字符串解析成元代码（CODE）；eval 执行元代码并返回结果。
函数获取与调用funcByName, makeCall, makeUnifiedCall, binaryExpr, unifiedExprfuncByName 按名称动态获取函数定义；makeCall / makeUnifiedCall 生成函数调用代码；binaryExpr / unifiedExpr 生成二元 / 统一调用形式的表达式。
SQL 元编程sql, sqlCol, sqlColAlias, sqlUpdate, sqlDelete动态生成 SQL 查询、列引用、列别名、UPDATE 和 DELETE 语句的元代码，通常配合 eval 执行。

二、高阶函数组（剔除系列函数）

组别核心类别对应函数作用内容
遍历与映射元素/列/行遍历each, eachLeft, eachRight, eachPre, eachPost, loop, ploop对向量、矩阵、表或字典按元素/行/列执行函数；each 为基础遍历，eachLeft / eachRight 处理二元函数两侧参数广播，eachPre / eachPost 用于前缀/后缀式序列化处理，loop / ploop 用于迭代计算，后者支持并行。
分解与规约结果聚合all, any, reduce将分解后的子结果再合并；all / any 常用于逻辑归约，reduce 用于把二元函数按序折叠为单值结果。
函数组合组合调用call, pcall, unifiedCall, compose, nothrowCallcall / pcall 对一组函数做顺序或并行调用；unifiedCall 用统一入参形式调用；compose 组合多个函数形成函数管道；nothrowCall 在调用失败时返回异常而不中断流程。
分组与切片分组聚合groupby, rowGroupby, segmentby, contextbygroupby 在分组变量上执行函数并返回每组结果；rowGroupby 面向行级矩阵/数组；segmentby 按技术信号分段；contextby 在组内上下文中逐行计算。
二维展开交叉与透视cross, pcross, pivot, byColumncross / pcross 生成笛卡尔组合并计算；pivot 将长表转换为矩阵式交叉表；byColumn 按列施加函数。
空值处理缺失值控制nullCompare, withNullFill控制高阶函数遇到 NULL 时的比较与填充行为，使结果更贴近 Python / 统计计算习惯。
