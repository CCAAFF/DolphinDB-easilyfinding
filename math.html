三角函数三角sin, cos, tan, asin, acos, atan正弦、余弦、正切及其反函数

集合运算集合intersection, symmetricDifference两集合的交集；对称差集（仅属于一方的元素）
矩阵构造构造与重排diag, tril, triu, repmat提取/构造对角矩阵；提取下/上三角矩阵；repmat 按指定次数重复平铺矩阵
矩阵分解分解cholesky, lu, qr, svd, eig, schurCholesky、LU、QR、SVD、特征值分解、Schur 分解，覆盖线性代数主流矩阵分解算法

向量运算范数与加权vectorNorm补, wsum2补vectorNorm 计算向量的 L1 / L2 / Lp 范数；wsum2 计算加权平方和 Σwᵢxᵢ²

统计（剔除系列的根函数）

描述统计、分位数、分布估计、数据变换、线性代数、指数加权、距离度量、插值拟合与上下文聚合。

组别核心类别对应函数作用内容
描述统计基本统计量mean, mode, rms, sem, maxIgnoreNull, minIgnoreNull, demean均值、众数、均方根、标准误差、忽略 NULL 的最大/最小值；demean 去均值（减去序列均值）
描述统计汇总概览summary, stat, crossStatsummary / stat 输出均值、标准差、分位数等描述统计表；crossStat 对两列或多列交叉计算统计量
描述统计变化率percentChange计算相邻元素或指定滞后期的百分比变化率
分位数分位计算quantile, quantileSeries, percentileRank, qcutquantile 计算单个分位数；quantileSeries 输出完整分位数序列；percentileRank 返回值在序列中的百分位排名；qcut 按分位数将数据分组
分布估计PDFpdfNormal补, pdfChiSquare补, pdfF补计算正态、卡方、F 分布的概率密度函数值
分布估计核密度估计gaussianKde, gaussianKdePredict高斯核密度估计拟合数据分布；gaussianKdePredict 对新点预测概率密度
分布估计频率统计histogram补, histogram2dhistogram 对一维数据按区间统计频数；histogram2d 对二维数据计算二维频率直方图
数据变换幂变换boxcoxBox-Cox 幂变换，将偏态分布的数据变换为近似正态分布
线性代数向量运算dot, wc, wcovar, atImax, atImindot 计算向量点积或矩阵乘法；wc 加权计数；wcovar 加权协方差；atImax / atImin 返回最大值/最小值位置对应的另一向量元素
指数加权EWM 统计ewmMean, ewmStd, ewmVar, ewmCov指数加权移动均值、标准差、方差、协方差，近期数据权重更高，适合金融时间序列平滑
距离度量多种距离mahalanobis补, minkowski补, seuclidean补Mahalanobis 距离（考虑协方差）；Minkowski 距离（可退化为欧氏或曼哈顿）；标准化欧氏距离
插值（补充）插值拟合cubicHermiteSplineFit, kroghInterpolateFit, linearInterpolateFit分段三次 Hermite、Krogh 多项式、线性插值的拟合接口，与插值表里的 cubicSpline、kroghInterpolate 形成「拟合 + 预测」配对
上下文聚合Group 内计算contextCount补, contextSum, contextSum2在 SQL context by 子句中对每组计算计数、求和、加权求和，结果按原始行对齐广播回去

相关性（选取系列的根函数）

线性相关（自相关）、非参数相关（秩相关/互信息）与距离相似度度量。

核心类别对应函数作用内容
线性相关autocorr, acf, ewmCorrautocorr 计算序列的自相关系数；acf 计算完整自相关函数（含滞后阶数）；ewmCorr 指数加权移动相关系数，对近期数据给予更高权重
非参数相关kendall, spearmanr, mutualInfoKendall τ 和 Spearman ρ 基于秩次度量单调相关性；mutualInfo 用信息熵衡量两变量间的非线性依赖程度
距离与相似度distance, euclidean, tanimoto, cosineeuclidean 计算欧氏距离；cosine 计算余弦相似度；tanimoto 计算 Tanimoto（Jaccard）相似系数；distance 提供多种距离度量的统一接口

序列分析

单调性检测、极值点检测与折线简化（ZigZag），常用于 K 线形态与信号提取。

核心类别对应函数作用内容
单调性检测isMonotonicIncreasing, isMonotonic, isMonotonicDecreasing检测序列是否单调递增、单调递减；isMonotonic 同时支持两种方向判断
极值点检测isPeak, isValley标记序列中的局部极大值（峰）和局部极小值（谷），常用于 K 线形态识别与信号提取
折线简化zigzag按指定幅度过滤噪声波动，提取序列中显著的高低转折点，生成 ZigZag 折线，常用于技术分析

数据处理

对象信息、拷贝转换、条件掩码、分桶索引、差分变化、分组累计、极值索引、窗口重采样、时间序列分解、降维标准化、字典更新与进化优化。

组别核心类别对应函数作用内容
对象信息维度与结构cols, rows, size, keys, values查询对象的列数、行数、元素数；获取字典或表的键集合与值集合
对象信息聚合判断all, any, countNanInf, isDuplicated, distinct, valueChanged判断是否全为真 / 任一为真；统计 NaN/Inf 个数；检测重复项、去重、检测值是否发生变化
拷贝与转换对象复制copy, deepCopy, asiscopy 浅拷贝对象；deepCopy 递归复制嵌套对象；asis 原样返回对象，阻止隐式类型转换
条件与掩码条件选择iif, mask, talibNulliif 向量化三目运算符；mask 按条件将元素置为 NULL；talibNull 将 0 替换为 NULL，兼容 TA-Lib 指标计算
分桶与索引分桶bucketCount, hashBucketbucketCount 统计各区间内元素数量；hashBucket 将值映射到指定数量的哈希桶编号
分桶与索引查找asof, lowerBoundasof 在已排序数组中找不大于目标值的最近位置；lowerBound 返回第一个不小于目标值的位置（二分查找）
差分与变化序列差分deltas, ratios, difference补, tmovedeltas 计算相邻元素差值；ratios 计算相邻元素比值；difference 求集合差集；tmove 按时间偏移移动序列
分组累计动态分组dynamicGroupCumcount, dynamicGroupCumsum在分组标签动态变化时，对每组计算累计计数 / 累计求和，适用于事件驱动分组场景
极值索引最大/最小位置imaxLast, iminLast, mimaxLast, miminLast, rowImaxLast, rowIminLast返回最后一个最大值/最小值的位置；m- 前缀为滑动窗口版本；row- 前缀为逐行版本
窗口与重采样时间窗口sessionWindow, sumbars, topRange补, resample, coeventsessionWindow 按会话间隔划分时间窗口；sumbars 按累计值确定 K 线根数；topRange 查找累计最大值范围；resample 时间序列重采样；coevent 检测两事件序列的共现关系
时间序列分解季节分解stl使用 STL 方法将时间序列分解为趋势、季节性和残差三个分量
降维与标准化数据变换pca, zscore, winsorize, winsorize!pca 主成分分析；zscore 标准化为均值 0 方差 1；winsorize / winsorize! 缩尾处理截断极端值，! 为原地修改版本
字典更新字典操作dictUpdate!原地将另一个字典的键值对合并更新到目标字典中，支持自定义冲突处理函数
进化优化全局搜索differentialEvolution差分进化算法，用于无梯度的全局最优化问题，适合含多个局部最优的复杂目标函数

插值

样条插值、多项式插值与局部加权回归（LOESS），用于离散点拟合与曲线平滑。

核心类别对应函数作用内容
样条插值cubicSpline, cubicSplinePredict, spline, splrep, splev, pchipInterpolateFit基于样条或分段三次 Hermite 函数拟合数据曲线；splrep 生成样条表示，splev 执行预测；cubicSpline / cubicSplinePredict 为三次样条的拟合与预测配对
多项式插值dividedDifference, kroghInterpolate, neville通过差商表（Newton 差商）、Krogh 公式或 Neville 迭代算法进行多项式插值，适合离散点拟合
局部加权回归loess对数据点做局部加权线性回归（LOESS / LOWESS），平滑散点噪声，常用于趋势估计

优化

无约束最小化、约束最优化、二次规划与线性规划，覆盖梯度/无梯度优化与投资组合优化常用求解器。

核心类别对应函数作用内容
无约束最小化fmin, fminBFGS, fminLBFGSB, fminNCG对目标函数进行无约束最小化；fmin 使用 Nelder-Mead 单纯形法；fminBFGS / fminNCG 使用梯度信息；fminLBFGSB 支持大规模变量并可加简单边界约束
约束最优化fminSLSQP, socp, qclp支持等式与不等式约束的非线性优化；fminSLSQP 使用序列最小二乘法；socp 为二阶锥规划；qclp 为二次约束线性规划
二次规划quadprog, osqp, scs求解二次目标函数 + 线性约束；osqp 和 scs 是大规模稀疏问题的高效求解器，常用于投资组合优化
线性规划linprog, solvelinprog 求解线性规划问题；solve 求解线性方程组 Ax = b
